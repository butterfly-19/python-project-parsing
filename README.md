# Python Project Parsing

Инструмент для структурного анализа многофайловых Python-проектов.

Проект рекурсивно обходит директорию, находит `.py` файлы, строит AST-дерево с помощью стандартного модуля `ast` и формирует JSON-описание структуры проекта.

## Возможности

- анализ многофайловых Python-проектов;
- извлечение импортов;
- извлечение функций и async-функций;
- извлечение классов и методов;
- определение декораторов;
- обработка аннотаций типов;
- извлечение docstring;
- поиск вызовов функций;
- поддержка современных конструкций Python:
  - lambda;
  - list/dict/set comprehensions;
  - generators;
  - try/except;
  - match/case;
  - f-strings;
  - async/await.

## Назначение проекта

Данный модуль разрабатывался как часть системы автоматической документации и анализа программного кода на основе LLM.

Результаты парсинга могут использоваться для дальнейшей передачи в:

- LLM-модель;
- векторную базу данных Qdrant;
- графовую базу данных Memgraph;
- систему генерации документации.
